## 📊 Student Performance Analysis (Descriptive Statistics)

This project focuses on analyzing student performance using descriptive statistics and visualization techniques.

### 🔍 Key Analysis:
- Data cleaning and understanding  
- Mean, Median, Mode  
- Skewness and distribution  
- Outlier detection using IQR  
- Gender-based comparison  
- Parental education impact  
- Correlation analysis  

### 📈 Key Insights:
- Reading and writing scores are higher than math  
- Math shows higher variability and more outliers  
- Students with higher parental education perform better  
- Strong correlation between reading and writing  

### 🛠️ Tools Used:
Python | Pandas | NumPy | Matplotlib | Seaborn
