# AI-Based Policy Analysis for Sugary Drink Consumption

Project Overview:
This project focuses on analysing the lack of effective policies addressing the overconsumption of sugary drinks in India. It uses Machine Learning to predict the impact of different policy measures on reducing consumption.

Objective:
-Analyse current policy gaps.
-Predict effectiveness of proposed policies.
-Support data-driven decision making.

Technology Used:
-Python
-Scikit-Learn
-TF-IDF Vectorisation
-Linear Regression
-Pandas,Numpy
-Matplotlib,Seaborn

Methodology:
-Text data processed using TF-IDF.
-Numerical features scaled using StandardScalar.
-Combined using ColumnTransformer.
-Model trained using Linear Regression.
-Evaluated using MSE and R² Score.

Results:
-Model successfully predicts consumption reduction.
-Strong Policies(tax+awareness+restrictions) show higher impact.
-Model performs reliably even with limited dataset.

Project Structure:
-Code
-Report
-Results

Report: The complete project report is available in the repository.

Future Scope:
-Use real_world datasets.
-Apply advanced ML Models.
-Build policy recommendation system.
