## 📊 Student Performance Analysis (Descriptive Statistics)

This project analyzes student performance using **descriptive statistics and data visualization** to uncover patterns and insights.

### 🔍 Key Analysis
- Data cleaning and understanding  
- Mean, Median, Mode  
- Distribution & Skewness  
- Outlier detection using IQR  
- Gender-based performance comparison  
- Impact of parental level of education  
- Correlation between subjects  

### 📈 Key Insights
- Students perform better in **reading and writing** than math  
- **Math shows higher variability** and more low-score outliers  
- A clear trend: **higher parental education → better performance**  
- Strong correlation between **reading and writing skills**  

### 🛠️ Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn
